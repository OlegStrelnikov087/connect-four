# Четыре в ряд (Connect Four)

Игра "Четыре в ряд" для двух игроков на React с TypeScript.

[Посмотреть мой сайт можно тут](https://os.seoadministrator.ru/)

## Технологии

- React 19
- TypeScript
- CSS3

## Установка и запуск

### Предварительные требования

- Node.js (версия 14 или выше)
- npm

### Запуск проекта

1. **Установить зависимости:**
    ```bash
   npm install


2. **Запустить проект в режиме разработки:**
    ```bash
   npm run dev

3. **Открыть браузер и перейдите по адресу:**
    http://localhost:5173

## Сборка проекта
    npm run build

## Просмотр собранной версии
    npm run preview

## Запуск тестов
    npm run test

## Тесты с покрытием
    npm run test:coverage

## Правила игры
Два игрока по очереди бросают фишки в колонки

Цель - собрать 4 фишки в ряд по горизонтали, вертикали или диагонали

Игрок 1 использует красные фишки

Игрок 2 использует желтые фишки

Игра заканчивается при победе одного из игроков или ничьей

## Управление
Нажмите на колонку, чтобы бросить фишку

Кнопка "Начать игру" - начать игру

Кнопка "Перезапустить" - начать новую игру

Кнопка "Выйти" - вернуться в главное меню

## Структура проекта

src/
- ui/    # React компоненты отвечающие за отображение в ui
- logic/ # кастомные хуки и функции отвечающие за логику
- tests/ # тесты 
- utils/ # вспомогательные функции, константы, типы, перечесления

## На что стоит обратить внимание?

- Разделение отображения от логики. Проект разбит на папки с ui (компоненты отвечающие за отображение интерфейса), utils (отдельные файлы с константами проекта, типами, перечислениями), logic (кастомные хуки и функции отвечающие за логику игры), test (папка с тестами)
- Задеплоиный проект с доп функционалом из ветки optional
