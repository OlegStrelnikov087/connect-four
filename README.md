# Четыре в ряд (Connect Four)

Игра "Четыре в ряд" для двух игроков на React с TypeScript.

[Посмотреть мой сайт можно здесь](https://os.seoadministrator.ru/)

## Технологии

- React 19
- TypeScript
- CSS3

## Установка и запуск

### Предварительные требования

- Node.js (версия 14 или выше)
- npm

### Запуск проекта

1. **Установить зависимости:**
    ```bash
   npm install


2. **Запустить проект в режиме разработки:**
    ```bash
   npm run dev

3. **Открыть браузер и перейдите по адресу:**
    http://localhost:5173

## Правила игры
Два игрока по очереди бросают фишки в колонки

Цель - собрать 4 фишки в ряд по горизонтали, вертикали или диагонали

Игра заканчивается при победе одного из игроков или ничьей или же игрок может досрочно завершить игру с помощью кнопки "Выйти"

## Управление
Чтобы изменить имя игрока, вам нужно в поле "Имя:" соответствующего игрока написать желаемое имя и нажать кнопку "Сохранить настройки"

Чтобы изменить цвет игрока, вам нужно из перечня цветов, которые будут в выпадающем списке, выбрать желаемый цвет и нажать кнопку "Сохранить настройки"

Нажмите на колонку, чтобы бросить фишку

Кнопка "Начать игру" - начать игру (если вы хотели изменить настройки игры которые стоят по умолчанию, то не забульте нажать кнопку "Сохранить настройки" перед запуском игры)

Кнопка "Перезапустить" - начать новую игру

Кнопка "Выйти" - вернуться в главное меню с настройкой игры
